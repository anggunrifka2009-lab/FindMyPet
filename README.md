# FindMyPet 🐾

FindMyPet adalah website adopsi hewan anjing dan kucing menggunakan laravel dan Tailwind CSS

## Fitur
- Login & Register
- Dashboard Admin
- Tambah Hewan
- Edit & Hapus Hewan
- Detail Hewan
- Adopsi via WhatsApp
